CSS-Cascading Style Sheets

CSS are 3 types

INLINE CSS , INTERNAL CSS , EXTERNAL CSS

INLINE CSS - Means for any html tag,writing CSS property inside the single tag itself.

INTERNAL CSS - Sheet may be used if one single HTML page has one unique style

EXTERNAL CSS - Sheet can be written in any text editor, and must be saved with a .css extension.
